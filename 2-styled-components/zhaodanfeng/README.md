## todos 启动说明

###### 1、下载代码到本地
<code>$ git clone https://github.com/SpencerZ/react-homework.git</code>

###### 2、进入我的文件夹
<code>$ cd 2-styled-components</code><br>
<code>$ cd zhaodanfeng</code>

###### 3、安装依赖包
<code>$ npm install</code>

###### 4、启动应用
<code>$ npm start</code>

## todos 功能特性

 * 可添加、完成、删除新任务
 * 可批量删除已完成的任务
 * 可筛选已完成、未完成和所有的任务
 * 显示未完成任务数量
